
# AppUpdate

An App Update Util for Android.

# Gradle

```
allprojects {
    repositories {
        jcenter()
        google()
        maven { url 'https://jitpack.io' }
    }
}
```

```
compile('com.github.paranoiaaa:AppUpdate:1.0.1') {
    exclude group: 'com.android.support'
}
```
